答：

```cpp
string s = "word";
string p1 = s + s[s.size() - 1] == 's' ? "" : "s";
```

因为优先级：下标运算符 > 加法运算符 > 相等运算符 > 条件运算符 > 赋值运算符，所以第二行的表达式实际的运算顺序是：

```cpp
string p1 = (((s + (s[s.size() - 1])) == 's') ? "" : "s");
```

`(s + (s[s.size() - 1]))`的结果是一个字符串，而`'s'`是一个字符，比较二者是否相等是非法的，所以无法通过编译。



应改成：

```cpp
string p1 = s + ((s[s.size() - 1] == 's') ? "" : "s");
```

以实现如果字符串s末尾无s则添加一个“s”，有则不变。
